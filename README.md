## 概要

[Qiita API v2](https://qiita.com/api/v2/docs) の `https://qiita.com/api/v2/items` エンドポイントより最新の記事一覧を取得する簡単な React アプリケーションです。記事タイトルには記事へのリンクが設定されています。

以下の要素を含みます：

* Hooks
  - useState
  - useEffect
* axios
* Bootstrap
